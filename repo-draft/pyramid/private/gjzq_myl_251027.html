1. 市场现状与风格变化

1.1 市场从震荡中恢复的风格均衡化

尊敬的各位投资者，当前市场正经历从震荡向均衡化风格的关键转变。

  • 本人作为国金证券策略分析师，今日就市场主线逻辑变化及中长期投资策略向各位汇报

  • 核心观察：市场已从前两周的剧烈震荡中逐步恢复，并呈现出明显的结构性变化

  • 风格转换：从单一风格主导转向多元化均衡配置阶段

1.2 TMT内部的结构性变化

TMT板块内部正在发生深刻的结构性轮动，呈现出从海外算力向国产算力转移的明确趋势。

  • 收益结构变化：收益率和成交占比已从光模块、PCB等海外算力产业链逐步转向国产算力领域

  • 补涨行情启动：前期表现相对弱势的信创、消费电子等细分领域开始出现回升迹象

  • 板块轮动逻辑：资金在不同TMT子板块间重新配置，寻求新的增长点

1.3 红利策略板块的突出表现

传统弱势板块在红利策略推动下创历史新高，彰显市场风格多元化特征。

  • 农业银行：突破历史高点，红利策略优势凸显

  • 中国神华：传统能源龙头在价值重估中表现突出

  • 宇通客车：制造业细分龙头在周期复苏中展现韧性

  • 投资启示：市场不再是单一风格主导，而是进入多元化配置阶段

1.4 风格均衡期的投资策略

在风格均衡化背景下，投资者应沿着未来主线进行前瞻性布局。

  • 核心判断：市场尚未出现大幅风格切换，而是进入风格均衡期

  • 投资逻辑：单纯的高低估值切换不会在短期内快速完成

  • 配置建议：逐步沿着未来主线，在具有明确行业逻辑的领域进行战略布局

  • 策略要点：避免追涨杀跌，关注基本面驱动的结构性机会

2. 全球经济结构分析

2.1 过去2-3年市场交易逻辑的演变

过去2-3年全球交易逻辑主要围绕服务业与制造业的分化展开，这一格局正面临重要拐点。

  • 交易主线：服务业持续强劲与制造业相对弱势的分化格局

  • 时间周期：这一分化格局已持续2-3年，超出历史常规周期

  • 结构变化：服务业和制造业相关数据即将在四季度前后进入新时期

  • 投资含义：原有交易逻辑需要重新评估和调整

2.2 服务业与制造业的分化趋势

美国经济呈现服务业持续荣枯、制造业陷入衰退的显著分化，这种背离现象具有历史独特性。

  • 服务业表现：持续保持在荣枯线上方，展现较强韧性

  • 制造业困境：意外跌破荣枯线，进入衰退区间

  • 政策影响：西方世界大规模财政转移支付政策是导致这种分化的关键因素

  • 历史意义：如此长时间的背离在历史上极为罕见

2.3 全球投资与贸易环境分析

全球投资持续低迷，贸易环境恶化，但中国出口企业展现出强劲竞争力。

  • 投资现状：

    • 全球固定资本形成总额持续下行

    • 投资水平处于历史低位区间

    • 制造业投资信心明显不足

  • 贸易挑战：

    • 全球贸易占GDP比重持续下降

    • 外需环境整体偏弱

    • 贸易增长动能不足

  • 中国优势：

    • 出口企业份额在疫情爆发后稳中有升

    • 竞争力指标高于2017年之前水平

    • 在全球贸易收缩中逆势提升市场份额

2.4 中国出口企业的竞争力与价格压力

中国出口企业在保持竞争力提升的同时面临显著的价格压力，这一矛盾现象反映了全球供需格局的深刻变化。

  • 竞争力提升：

    • 产能投资大幅增加

    • 技术升级成效显著

    • 生产线设备达到国际先进水平

  • 价格压力加剧：

    • 2023-2024年出口价格指数大幅下跌

    • PPI持续下行压力增大

    • 企业盈利空间被压缩

  • 历史脉络：

    • 2021年：限电限产政策影响

    • 2022年：内需不足成为主要制约

    • 2023年：出口价格下跌成为新挑战

2.5 全球供需失衡的结构性分析

中国作为全球工厂，面对的是贸易需求偏弱与供给不断增加的结构性矛盾。

  • 需求端疲软：全球贸易需求相对偏弱，复苏动力不足

  • 供给端扩张：中国制造业产能持续扩张，供给能力不断增强

  • 价格传导机制：商品通胀相对于服务业通胀持续下行

  • 结构性影响：制造业偏弱、服务业偏强的格局形成

3. 金融资产与实体经济的关系

3.1 大宗商品与股市表现分化

大宗商品与股市表现出现显著分化，中国制造业内卷与海外市场疲软形成鲜明对比。

  • 全球大宗商品：自2022年底以来表现持续疲弱

  • 中国特殊情况：由于制造业内卷，大宗商品在中国市场走出红利属性

  • 周期性复苏：2024年一季度至二季度有色金属出现短暂上涨

  • 制约因素：服务业过强导致降息迟滞，制造业复苏难以兑现

3.2 服务业强势对制造业复苏的制约

服务业的过度强势成为制造业复苏的重要制约因素，通过降息预期影响实体经济。

  • 制约机制：服务业强劲表现延缓降息节奏

  • 复苏困难：制造业复苏需要更宽松的货币政策环境

  • 金融资产影响：以美股为代表的全球金融资产在这种环境下表现突出

  • 结构性问题：服务强、制造弱的失衡局面持续存在

3.3 全球金融资产估值的历史性高位

全球金融资产估值已达到历史极端高位，金融资产占GDP比例突破3年两倍标准差。

  • 估值水平：全球金融资产占GDP比例创历史新高

  • 包含范围：涵盖6个主要国家股市及数字货币

  • 中国贡献：2024年5月以来中国资产推动了最后的上涨斜率

  • 资产类比：

    • 中国AI相关资产：类似白银角色

    • 海外科技股：类似黄金角色

    • 卫星资产：展现出更大弹性

3.4 中美科技资产的关联性分析

中美科技资产呈现高度关联性，中国科技股的弹性特征在市场波动中表现明显。

  • 关联机制：中国AI资产与海外科技股存在密切关联

  • 表现差异：2024年4-5月中国AI资产涨幅超过美股整体

  • 弹性特征：作为卫星资产，中国科技股在调整中跌幅更大

  • 市场结构：美股大盘股持续震荡上行，中小盘股回报率开始下行

3.5 金融市场结构变化与风险预警

金融市场出现"抢蛋糕"现象，企业盈利增长动能放缓，估值扩张面临挑战。

  • 市场结构变化：

    • 美股大盘股回报率震荡上行

    • 中小盘股回报率开始下行

    • 市场进入存量博弈阶段

  • 盈利增长挑战：

    • 科技企业盈利进一步上升需要估值扩张

    • 金融资产占GDP比例触及历史两倍标准差

    • 大企业挤压小企业盈利空间

  • 风险因素：

    • 服务业走弱带来金融资产波动

    • 环境可持续性担忧加剧

    • 二阶导数层面出现风险信号

3.6 企业盈利增长动能的放缓趋势

企业盈利增长动能明显放缓，大企业挤压小企业盈利的模式难以持续。

  • 增长瓶颈：科技企业盈利增长需要估值进一步扩张

  • 融资需求：资本开支需要股权或债权融资支撑

  • 估值约束：金融资产估值触及历史高位，扩张空间有限

  • 盈利分配：大企业抢夺小企业盈利，整体增长动能放缓

4. 制造业复苏的投资机会

4.1 美国制造业周期性拐点分析

美国制造业正从衰退走向复苏，但服务业与制造业的周期位置差异带来复杂局面。

  • 制造业复苏信号：

    • 美国制造业可能从衰退走向复苏

    • 商品通胀相对上浮位开始回弹

    • 制造业修复动能已经启动

  • 服务业不确定性：

    • 服务业处于过热滞涨期

    • 面临转向衰退的风险

    • 走弱可能导致金融资产大幅波动

  • 两种路径：

    • 危机模式：服务业快速转衰退

    • 平稳过渡：渐进式调整

4.2 经济复苏的两种可能路径

经济复苏存在危机模式和平稳过渡模式两种可能路径，需要做好风险防范准备。

  • 危机模式特征：

    • 服务业快速转向衰退

    • 金融资产出现大幅波动

    • 市场信心受到严重冲击

  • 平稳过渡特征：

    • 服务业渐进式调整

    • 金融市场相对稳定

    • 经济结构逐步优化

  • 当前判断：未来一段时间可能进入阶段性平稳过渡模式

  • 风险提示：危机风险不可不防，需要保持警惕

4.3 制造业修复动能的先行指标

多项先行指标显示制造业修复动能正在积聚，降息周期预期为制造业复苏创造条件。

  • 价格信号：美国商品通胀相对上浮位开始回弹

  • 周期指标：降息周期指标领先制造业周期指数约6个月

  • 修复进度：约70%经济体已进入降息周期，制造业修复动能已在路上

  • 数据验证：最新制造业数据显示向上趋势

4.4 中国出口价格指数的见底回升

中国出口价格指数接近历史底部，预示着制造业复苏和通缩缓解的重要信号。

  • 指数走势：中国出口价格指数接近100关口

  • 时间起点：从去年四季度开始出现回升迹象

  • 驱动因素：

    • 中国反内卷政策开始见效

    • 制造业新一轮小型上行周期启动

  • 历史意义：过去10年来这一信号具有重要预示作用

4.5 出口价格指数与PPI的历史关联性

出口价格指数与PPI存在历史性关联，当前环境与2010-2017年时期具有相似性。

  • 历史对比：2010-2017年出口价格指数与PPI关系较为一致

  • 当时背景：

    • 大规模基础设施建设时期

    • 通胀有所回落

    • 出口出现增收不增利现象

  • 当前启示：

    • 海外需求恢复将推动出口价格指数率先见底

    • 随后带动PPI整体反弹

    • 中国有望走出通缩困境

4.6 海外需求恢复与国内供应结构调整

海外需求恢复与国内供应结构调整形成共振，为中国经济复苏创造有利条件。

  • 需求端改善：海外需求逐步恢复

  • 供给端优化：中国供应结构持续调整

  • 价格传导：出口价格指数率先见底回升

  • 宏观影响：PPI层面走出通缩可以期待

  • 微观机制：中国工厂外销产品价格止跌，内部价格逐级下跌趋势有望缓解

4.7 中国通缩环境的结构性成因

中国通缩环境具有深刻的结构性成因，资本开支大幅收缩为未来复苏创造条件。

  • 资本开支现状：

    • 资本开支创15年新低

    • 资本开支比折旧摊销处于低位

    • 固定资产投资增速放缓

  • 企业盈利改善：

    • 制造业利润率开始回升

    • 工业企业盈利状况改善

    • 周转率成为当前短板

  • 核心矛盾：

    • 过去2-3年GDP未明显下跌

    • 但企业盈利持续下行

    • 体现了单位GDP盈利下降的"实物消耗"现象

4.8 企业盈利修复的微观基础

企业盈利修复的微观基础正在形成，利润率回升为复苏奠定基础。

  • 利润率表现：制造业利润率已回升至2023年水平附近

  • 结构性变化：

    • GDP修复动能有限

    • 但企业盈利进入修复空间

    • 利润率与周转率双重改善预期

  • 理论支撑：与"实物消耗"理论的镜像效应一致

  • 投资含义：盈利修复将成为市场主线之一

4.9 利润率回升与周转率改善预期

企业利润率已明显回升，周转率改善将成为下一阶段盈利修复的关键驱动力。

  • 当前状况：利润率回升至2023年附近水平

  • 改善空间：周转率仍有较大提升空间

  • 双重驱动：

    • 海外需求上行提供外部支撑

    • 国内投资放缓创造内部条件

  • 盈利前景：企业盈利有望陆续进入修复通道

5. 新一轮制造业复苏的驱动因素

5.1 AI产业对电力需求的拉动作用

人工智能产业正在成为新一轮制造业复苏的重要驱动力，电力需求激增成为重要信号。

  • 电力市场表现：

    • 美国平均电价创2025年以来历史新高

    • 电力供应出现明显紧张状况

    • AI投资与电力需求增长高度相关

  • 扩散效应：AI投资从算力领域向基础设施领域扩散

  • 宏观影响：推动全球最大经济体开启建设周期

5.2 AI投资从算力向基础设施的扩散

AI投资正在从核心算力向更广泛的基础设施领域扩散，创造新的投资机会。

  • 扩散路径：从直接相关的算力设备向基础设施延伸

  • 投资逻辑：AI发展需要更完善的基础设施支撑

  • 市场机会：基础设施领域成为新的投资热点

  • 产业影响：推动相关制造业需求增长

5.3 全球电力短缺与中国光伏出口机遇

全球电力短缺为中国光伏出口创造重要机遇，中国电力优势成为核心竞争优势。

  • 全球电力短缺格局：

    • 非洲、拉美、东南亚人均发电量相对较低

6. 投资主线选择的理性思考

6.1 热门板块与实际收益的背离

热门板块与实际收益存在明显背离，投资者需要理性看待市场热点。

  • 历史经验：2021年10-11月之后的市场表现提供重要参考

  • 对比案例：2022年以来煤炭石油石化火电表现优于新能源

  • 基本面与股价背离：

    • 新能源新增装机容量持续增长

    • 光伏装机到2023年都在上行

    • 但相对收益从2023年开始跑输传统能源

  • 投资启示：基本面拐点与股价拐点关系复杂，需要理性分析

6.2 新能源行业景气度与股价表现差异

新能源行业景气度与股价表现出现显著差异，反映了市场预期的复杂性。

  • 景气度数据：

    • 新增装机容量持续增长

    • 光伏装机到2023年都在上行

    • 2024年大部分时间保持高景气状态

  • 股价表现：

    • 2023年开始跑输煤炭电力

    • 2024年逐步跑输

    • 相对收益持续下滑

  • 背离原因：市场预期、估值水平、资金偏好等多重因素影响

6.3 基本面拐点与股价拐点的复杂关系

基本面拐点与股价拐点存在复杂关系，争议增大不一定意味着行业反转。

  • 复杂性体现：基本面拐点与股价拐点并非简单对应关系

  • 争议的意义：新兴科技领域争议增大不必然导致行业反转

  • 投资逻辑：当行业景气未反转时，新的投资机会可能已经在其他领域出现

  • 当前判断：新的投资机会正在其他领域逐步体现
