# Text Enhance Persuasive Command

基于局部论述结构识别方法，增强文本的论述结构表达，使论证方法更加明确和有说服力。

## Usage

```markdown
/text-enhance-pursuasive "输入文本"
```

## Parameters

- `input_text`: 需要增强论述结构的文本内容
- **注意**: 仅改写章节内容，章节标题保持不变

## Output

该命令将论述结构增强后的文本直接输出到控制台，不会修改输入文件或创建新文件。

## 核心功能

### 1. 局部论述结构识别

#### 自动识别18种局部论述结构
**基础逻辑类结构识别：**
- **逻辑推理结构**：识别"因为...所以..."、前提与结论关系
- **因果分析结构**：识别原因探讨、因果链条
- **反证法结构**：识别假设相反、排除法论证
- **逻辑结构清晰结构**：识别层次分明的论证框架

**证据支撑类结构识别：**
- **具体例子支撑结构**：识别"例如"、"比如"等例证表达
- **数据统计支撑结构**：识别数字、百分比等量化证据
- **研究引用支撑结构**：识别研究机构、学者、报告等权威引用
- **案例实例支撑结构**：识别历史事件、现实案例等实例支撑

**对比分析类结构识别：**
- **对比分析结构**：识别对比对象、差异性分析
- **类比论证结构**：识别相似性比较、类比推理

**权威支撑类结构识别：**
- **权威引用结构**：识别权威人士或机构的观点引用
- **专业观点引用结构**：识别专业理论、学术概念的引用

**前瞻性分析类结构识别：**
- **预测未来结构**：识别未来趋势判断、预测分析
- **背景趋势引用结构**：识别宏观背景、发展趋势引用

**价值理念类结构识别：**
- **道德伦理论证结构**：识别价值观、道德标准判断

**修辞表达类结构识别：**
- **修辞手法结构**：识别比喻、排比、反问等修辞表达
- **情感诉求结构**：识别情感表达、价值观诉求

**论证完善类结构识别：**
- **反驳反对意见结构**：识别反对观点、反驳论证

### 2. 三步识别法应用

#### 第一步：关键词扫描
- **逻辑连接词扫描**：因为、所以、由于、因此、首先、其次、最后
- **证据标志词扫描**：例如、比如、数据显示、研究显示、专家认为
- **对比标志词扫描**：相比之下、与此不同、同样地、类似于
- **权威标志词扫描**：权威机构、专家、研究、报告、理论

#### 第二步：结构模式匹配
- **结构起点识别**：段首论点句或主题句
- **发展脉络追踪**：论证过程中的逻辑推进
- **结构终点确认**：段尾结论句或总结句

#### 第三步：功能效果评估
- **论证强度评估**：证据充分性、逻辑严密性
- **结构效果评估**：表达清晰度、说服力强度
- **整体贡献评估**：对全文论证的作用意义

### 3. 论述结构增强策略

#### 显性化表达增强
**原文本可能存在的隐含结构：**
```
隐含表达：人工智能发展很快，很多公司都在投资，应用也很多。

增强后：**人工智能技术近年来发展迅速，主要体现在企业投入和应用场景两个维度。**
- **企业投入增加**：众多科技公司持续加大研发资源投入
- **应用场景扩展**：从智能驾驶到医疗诊断的广泛应用
```

#### 凸显词汇应用
**结构化标识词汇：**
- **逻辑推理类**：基于...可以推断出、进一步推理、由此可见
- **因果分析类**：究其原因、这导致了...、根本原因是...
- **例证支撑类**：以...为例、这个例子清楚地表明...、充分证明了...
- **数据支撑类**：数据显示...、具体来看...、这些数字充分说明...
- **权威引用类**：根据...研究显示、专家认为...、研究结果证实了...
- **对比分析类**：相比之下、前者...而后者...、因此...更具有优势
- **类比论证类**：正如...一样、同样地...、这个类比告诉我们...

### 4. 改写增强原则

#### 必须保持的内容
- ✅ **所有原始论点**：核心观点和分论点完整保留
- ✅ **所有论证材料**：例子、数据、引用、案例一个不缺
- ✅ **原文本意图**：作者的表达意图和论证目标
- ✅ **信息准确性**：事实、数据、引用的准确性

#### 必须增强的要素
- ✅ **论述结构显性化**：明确标识论证方法类型
- ✅ **逻辑关系清晰化**：用词汇展现论证的逻辑推进
- ✅ **层次结构分明化**：用词汇区分论证的不同层次
- ✅ **表达效果强化**：增强说服力和表达清晰度

#### 改写质量标准
- **结构识别准确率**：100%准确识别原文论述结构
- **增强效果显著性**：论述结构明显凸显出来
- **表达自然度**：增强后的表达自然流畅
- **论证完整性**：不破坏原有的论证完整性

## 处理流程

### 第一阶段：结构分析识别
1. **文本预处理**：分句、分段，识别基本结构
2. **关键词扫描**：扫描逻辑连接词、证据标志词、对比标志词等
3. **结构模式匹配**：匹配18种局部论述结构模式
4. **功能效果评估**：评估现有结构的表达效果

### 第二阶段：增强策略制定
1. **识别隐含结构**：发现没有明确标识的论述结构
2. **制定增强方案**：为每个识别的结构制定增强策略
3. **选择凸显词汇**：根据结构类型选择合适的凸显词汇
4. **设计改写框架**：保持原意的基础上设计改写框架

### 第三阶段：文本增强改写
1. **结构显性化**：用明确词汇标识论述结构
2. **逻辑清晰化**：强化逻辑关系的表达
3. **层次分明化**：增强论证层次的清晰度
4. **效果优化**：确保增强后的自然度和说服力

## 增强示例

### 示例1：逻辑推理结构增强
**原文本：**
```
人工智能发展很快，因为算法进步了，所以应用也多了。
```

**增强后：**
```
**基于算法进步的根本推动，人工智能技术的应用场景得到了显著扩展。**
- **逻辑推理分析**：算法的持续进步直接推动了技术能力的提升
- **推理结果验证**：因此，人工智能在各行各业的应用广度和深度都得到了显著增长
- **结论**：由此可见，算法发展是人工智能应用普及的核心驱动力
```

### 示例2：例证支撑结构增强
**原文本：**
```
很多企业都在用人工智能，比如电商推荐，还有自动驾驶。
```

**增强后：**
```
**人工智能技术已经在多个行业实现了广泛应用，其中电商推荐和自动驾驶是典型例证。**
- **具体例证一**：以电商平台为例，个性化推荐系统显著提升了用户体验和销售转化率
- **例证分析**：这个例子清楚地表明人工智能在商业领域的实际价值
- **具体例证二**：自动驾驶领域同样展现了人工智能的变革性影响
- **例证总结**：这些典型案例充分证明了人工智能技术应用的广泛性和有效性
```

### 示例3：对比分析结构增强
**原文本：**
```
传统方法效率低，人工智能方法效率高。
```

**增强后：**
```
**通过对比分析可以发现，人工智能方法相比传统方法具有显著的效率优势。**
- **对比对象一**：传统方法往往依赖人工操作，存在效率低下和易出错的局限性
- **对比对象二**：人工智能方法通过算法自动化处理，实现了效率的大幅提升
- **差异分析**：前者在处理速度和准确性方面明显落后于后者
- **对比结论**：因此，人工智能方法在效率方面具有明显的竞争优势
```

### 示例4：因果分析结构增强
**原文本：**
```
技术发展了，所以生活变好了。
```

**增强后：**
```
**深入分析可以发现，技术发展是推动生活质量提升的根本原因。**
- **因果链条分析**：技术的不断进步为社会发展提供了强大的动力支撑
- **深层原因探究**：究其原因，技术创新在多个维度改善了人们的生活条件
- **结果呈现**：这导致了生活方式的便利化、工作效率的提升以及服务质量的改善
- **根本结论**：因此，技术发展是生活质量持续提升的核心驱动力
```
