# Text Enhance Persuasive Command

对文本进行简洁化和清晰化改写，优化表达方式，增强文本的说服力和可读性。

## Usage

```markdown
/text-enhance-pursuasive "输入文本"
```

## Parameters

- `input_text`: 需要改写的文本内容
- **注意**: 仅改写章节内容，章节标题保持不变

## Output

该命令将改写后的文本直接输出到控制台，不会修改输入文件或创建新文件。

## 核心功能

### 1. 文本表达优化
- **逻辑关系识别**：识别因果、对比、举例、递进等逻辑关系
- **表达清晰化**：调整语序和用词，使表达更加准确
- **结构优化**：优化句子结构，增强可读性和说服力

### 2. 局部论述结构凸显
- **论述结构识别**：识别文本中的逻辑推理、因果分析、例证支撑等结构
- **凸显词汇应用**：使用专门的凸显词汇明确标识论述方法
- **表达层次化**：通过词汇区分不同类型的论证逻辑

### 3. 改写策略
- **适度凸显**：每句使用1-2个凸显词汇，明确论述结构
- **保持原意**：严格保持原文信息，零添加零删减
- **优化表达**：调整语序和用词，使表达更清晰自然
- **避免过度**：不使用生硬的学术化表达，保持流畅性

### 4. 论述结构凸显词汇库

#### 逻辑推理类凸显词汇
- **基于**...可以推断出、**因此**、**由此可见**、据此可证、进一步推理

#### 因果分析类凸显词汇
- **究其原因**、**导致**...、**根本原因是**...、**使得**、**因为**、**推动**、由此造成

#### 例证支撑类凸显词汇
- **以**...**为例**、**这个例子清楚地表明**...、**充分证明了**...、比如、具体来看

#### 数据支撑类凸显词汇
- **数据显示**...、**统计表明**...、**这些数字充分说明**...、具体数据为、量化分析显示

#### 引用支撑类凸显词汇
- **根据**...**研究显示**、**专家认为**...、**研究结果证实了**...、文献表明、学术指出

#### 对比分析类凸显词汇
- **相比之下**、**前者**...**而后者**...、**更具有优势**、与此不同、同样地

#### 类比论证类凸显词汇
- **正如**...**一样**、**同样地**...、**这个类比告诉我们**...、类似于、好比

#### 权威引用类凸显词汇
- **正如**...**所言**、**权威机构指出**、**具有权威性**、官方数据显示、权威观点认为

### 5. 质量控制原则

#### 必须保持的内容
- ✅ **核心观点**：原文的主要论点和分论点
- ✅ **关键信息**：重要的数据、例子、引用
- ✅ **原作意图**：作者想要表达的核心意思
- ✅ **事实准确性**：所有事实性信息的准确性

#### 优化的重点
- ✅ **适度凸显**：用少量凸显词汇明确论述结构
- ✅ **表达清晰度**：使表达更加明确易懂
- ✅ **信息完整性**：严格保持原文信息，零增删
- ✅ **自然流畅**：确保改写不生硬，保持易读性

## 处理流程

### 第一步：文本分析
1. **识别逻辑关系**：分析文本中的因果、对比、举例等关系
2. **理解核心观点**：把握作者想要表达的主要意思
3. **评估表达效果**：判断现有表达的清晰度和说服力

### 第二步：制定改写方案
1. **确定优化重点**：识别需要改进的表达问题
2. **选择改写策略**：根据文本特点选择合适的改写方法
3. **设计改写框架**：在保持原意基础上设计改写方案

### 第三步：执行改写
1. **优化表达方式**：调整语序、更换词汇、重组句子
2. **强化逻辑关系**：用恰当的连接词强化逻辑
3. **提升说服力**：选择更有力的表达方式
4. **确保质量**：检查改写效果和准确性

## 增强示例

### 示例1：逻辑推理结构增强
**原文本：**
```
人工智能发展很快，因为算法进步了，所以应用也多了。
```

**增强后：**
```
**基于**算法进步推动应用增多，**可以推断出**人工智能发展迅速。
```

### 示例2：例证支撑结构增强

**原文本：**
```
很多企业都在用人工智能，电商推荐、自动驾驶都在广泛使用。
```

**增强后：**
```
人工智能在众多企业得到广泛应用，**例如**电商推荐和自动驾驶。
```

### 示例3：对比分析结构增强

**原文本：**
```
传统方法效率低，人工智能方法效率高。
```

**增强后：**
```
**相比之下**，人工智能方法效率高，**而**传统方法效率较低**。
```

### 示例4：因果分析结构增强

**原文本：**
```
技术发展了，生活变好了。
```

**增强后：**
```
**技术进步**使得**生活质量得到显著提升。
```

### 示例5：数据支撑结构增强

**原文本：**
```
该功能具有广泛用户基础，研究表明80%的用户喜欢这个功能
```

**增强后：**
```
**研究显示**，80%的用户偏好此功能，**充分说明**该功能具有广泛的用户基础。
```

### 示例6：权威引用结构增强
**原文本：**
```
专家说AI很重要，权威机构认为 AI 发展将深刻影响未来社会
```

**增强后：**
```
**专家表示**，人工智能技术很重要，**权威机构指出**AI发展将深刻影响未来社会。
```

## AI执行提示词

```text
你是一位专业的文本表达优化专家，请对输入文本进行适度改写，在保持原意完整的基础上，通过语义识别论述结构，并使用恰当的凸显词汇来明确这些结构，使表达更加清晰自然。

**核心原则：**
- **信息零添加零删减**：严格保持原文所有信息完整
- **语义识别优先**：通过语义内容判断论述结构，不依赖标志词语
- **适度凸显**：每句最多使用1-2个凸显词汇，避免过度使用
- **表达流畅**：确保改写后的文本自然易读，不生硬

**核心任务：**
1. **通过语义分析**识别文本中的论述结构类型
2. **自主选择或创造**合适的凸显词汇进行适度标识
3. 优化句子结构，使表达更清晰流畅
4. 输出保持原意、适度凸显的改写文本

**论述结构语义识别方法：**

1. **逻辑推理结构**：
   - **语义识别**：识别前提与结论的推导关系，即使没有"因为所以"
   - **改写词汇参考**：基于...可以推断出、因此、由此可见、由此可证
   - **自主选择**：可使用类似词汇如：据此可知、由此可见、这表明

2. **因果分析结构**：
   - **语义识别**：识别原因与结果的内在联系，不论是否有关联词
   - **改写词汇参考**：因为、使得、导致、究其原因、根本原因是
   - **自主选择**：可使用类似词汇如：造成了、促成了、源于、由于

3. **例证支撑结构**：
   - **语义识别**：识别具体事例对观点的支撑作用
   - **改写词汇参考**：例如、以...为例、充分说明、这个例子表明
   - **自主选择**：可使用类似词汇如：比如、具体来看、举例来说、由此可见

4. **数据支撑结构**：
   - **语义识别**：识别统计数据、数字信息对观点的支撑
   - **改写词汇参考**：研究显示、数据显示、统计表明、这些数字说明
   - **自主选择**：可使用类似词汇如：据统计、量化分析显示、数据为证

5. **引用支撑结构**：
   - **语义识别**：识别权威观点、研究成果对论证的支撑
   - **改写词汇参考**：专家认为、根据研究、文献表明、研究证实
   - **自主选择**：可使用类似词汇如：学者指出、学术观点认为、文献记载

6. **对比分析结构**：
   - **语义识别**：识别两个或多个对象的差异性比较
   - **改写词汇参考**：相比之下、而、前者...后者、不同在于
   - **自主选择**：可使用类似词汇如：与此不同、与此相反、另一方面

7. **类比论证结构**：
   - **语义识别**：识别基于相似性的推理和比较
   - **改写词汇参考**：正如...一样、同样地、如同、类似于
   - **自主选择**：可使用类似词汇如：好比、类似于、与...相似、同样

8. **权威引用结构**：
   - **语义识别**：识别权威人士、机构的观点和立场
   - **改写词汇参考**：权威机构指出、专家表示、正如...所言
   - **自主选择**：可使用类似词汇如：官方数据显示、权威观点认为、学界公认

**重要改写原则：**

1. **语义识别优先**：
   - 通过句子的语义内容判断论述结构，不依赖表面关键词
   - 理解作者的真实论证意图和逻辑关系
   - 准确识别隐含的论述结构

2. **灵活使用凸显词汇**：
   - 参考词汇库中的词汇，但可根据语境自主选择类似表达
   - 鼓励创造更贴切、更自然的凸显词汇
   - 避免机械套用，要符合具体语境

3. **信息完整性**：
   - 绝对不添加原文中没有的信息
   - 绝对不删减原文中的任何信息
   - 保持所有数据、例子、观点完整

4. **表达流畅性**：
   - 确保改写后语句通顺自然
   - 避免生硬的学术化表达
   - 保持原文的语言风格

**输出要求：**
- 直接输出改写后的章节内容
- 保持标题结构不变，只改写章节内容
- 基于语义识别选择恰当的凸显词汇
- 严格保持原文信息完整，零添加零删减
- 不包含其他解释，只输出改写后的文本

**质量标准：**
- 原文信息100%保留，无任何增删
- 通过语义准确识别论述结构
- 凸显词汇使用灵活且恰当
- 表达清晰流畅，不生硬
- 论述结构适度明确但不突兀
```
