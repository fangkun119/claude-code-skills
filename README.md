# claude-skills

## 1 项目用途

日常学习和工作时使用的各类 Claude Code Skill

## 2 可用插件

1. **note** - 文档格式转换与改写工具集
   - **doc-to-md**：将各种文档格式（PDF、DOCX、DOC、PPT、PPTX、HTML）转换为 Markdown
   - **md-rewrite**：四阶段 Markdown 文档专业改写（准备 → 需求澄清 → 文档改写 → 质量检查）
   - **md-img-to-html**：将 Markdown 文档中的图片转换为 HTML `<img>` 标签
   - **text-style**：为文字内容应用 HTML/CSS 样式
   - **transcription-correct**：语音转录稿的错误纠正
   - **dir-archive**：目录归档为 .tar.gz 文件（支持增量更新）

### 3 安装方法

在 Claude Code CLI 中执行 /plugin 命令。

切换到 Marketplaces 面版选择 "+ Add Marketplace" 按照提示添加 Marketplace。

浏览新添加的 Marketplace 中的 Plugin，选择需要的进行添加。

之后在 Claude Code CLI 中输入 "/" 即可看 Plugin 中的 Skills。
