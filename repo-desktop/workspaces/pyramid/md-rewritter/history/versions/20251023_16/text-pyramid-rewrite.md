# Pyramid Rewrite Command V2

基于金字塔原理和论述结构识别的智能文本重写工具，保持核心功能的同时简化复杂度，提升处理效率。

## Usage

```markdown
/text-pyramid-rewrite "章节内容文本" "额外需求（可选）"
```

## Parameters

- `text`: 需要重写的章节内容
- `requirements`: 额外需求（可选，默认空字符串）

## Output

重写后的内容以纯文本格式直接输出到控制台。输出采用UTF-8编码，确保与所有文本编辑器和终端兼容。

## 核心功能

### 1. 金字塔原理改写
- **智能结论**：重要段落开头用**加粗**显示核心结论，避免过度格式化
- **灵活列表**：根据内容性质决定是否转换为列表，建议适度列表化
- **逻辑层次**：保持核心结论→主要论据→具体例证的自然层次结构

### 2. 论述结构识别与自然化保留
- **智能识别**：自动识别8种核心论述结构
- **自然化保留**：将论述结构词汇自然融入文本，避免生硬表达
- **逻辑保护**：保持原有论证逻辑关系和说服力

### 3. 核心论述结构与凸显词汇

#### 逻辑推理类
- **逻辑推理**：基于...可以推断出、因此、由此可见
- **因果分析**：究其原因、这导致了...、根本原因是...

#### 证据支撑类
- **例证支撑**：以...为例、这个例子清楚地表明...、充分证明了...
- **数据支撑**：数据显示...、统计表明...、这些数字充分说明...
- **引用支撑**：根据...研究显示、专家认为...、研究结果证实了...

#### 对比分析类
- **对比分析**：相比之下、前者...而后者...、更具有优势
- **类比论证**：正如...一样、同样地...、这个类比告诉我们...

#### 权威支撑类
- **权威引用**：正如...所言、权威机构指出、具有权威性

## 处理流程

### 三步简化流程
1. **识别分析**：扫描原文，识别论述结构和凸显词汇
2. **结构重构**：按金字塔原理重组内容，保留关键词汇
3. **质量优化**：确保逻辑清晰、信息完整、表达流畅

### 智能内容处理
- **论述性内容**：保留逻辑推理和证据支撑方式
- **叙事性内容**：保持时间顺序，保留关键事件
- **说明性内容**：规范步骤说明，保持逻辑清晰
