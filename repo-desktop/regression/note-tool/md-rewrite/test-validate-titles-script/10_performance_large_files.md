## 预期结果
- 性能表现：
  - 脚本应在合理时间内处理完所有内容（< 5秒）
  - 内存使用应保持稳定，不应出现内存泄漏
  - 处理时间应与文档大小呈线性关系
- 大量标题：
  - 应正确识别所有标题，不遗漏任何标题
  - 编号连续性检查应准确无误
- 大量代码块：
  - 所有代码块内容都应被正确跳过
  - 代码块处理不应影响性能
- 混合内容：
  - 标题和代码块的混合场景应被正确处理
  - 代码块边界识别应准确无误
- 边界情况：
  - 超长标题应被正确处理，不应截断或报错
  - 深层嵌套结构应被正确解析
- 内存效率：
  - iter_code_aware_lines() 函数应有效使用内存
  - 不应将整个文件内容一次性加载到内存中
